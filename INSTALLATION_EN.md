# Installation of Contao Grid-16-Columns Bundle

There are two types of installation.

* with the Contao-Manger, for Contao Managed-Editon
* via the command line, for Contao Managed-Editon


## Installation with Contao-Manager

* Search for package: `bugbuster/contao-grid_16-bundle`
* Install the package
* Update the database


## Installation via command line

### Installation for Contao Managed-Edition

Installation in a Composer-based Contao 4.13+ / 5.3+ Managed-Edition:

* `composer require "bugbuster/contao-grid_16-bundle"`
* `php bin/console contao:migrate`
